### Power Draw and Estimated Runtime
Based on a standard 10,000mAh (37Wh) powerbank at 85% conversion efficiency (~31.45Wh usable).

| Configuration         | Current Draw  | Power @ 9V | Estimated Run Time |
| --------------------- | ------------- | ---------- | ------------------ |
| **Version 1 (GP-5)**  | 600 mA        | 5.4 Watts  | **~5.8 Hours**     |
| **Version 2 (GP-50)** | 1,000 mA (1A) | 9.0 Watts  | **~3.5 Hours**     |

Note: The GP-50 includes a built-in 1450 mAh battery that acts as a fail-safe, providing approximately 2.9 hours of independent operation for the multi-effects unit.
